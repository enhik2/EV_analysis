Визуальный анализ данных с помощью Python (numpy, pandas, matplotlib, seaborn).
Мировые продажи электромобилей: 2010-2024.


Данные: https://www.kaggle.com/datasets/patricklford/global-ev-sales-2010-2024
Глобальные данные по электромобилям: 2010-2024 гг., с некоторыми прогнозами на 2025-2035 гг. (Китай, Северная Америка, другие (сгруппированные регионы) и мир).

Чтобы установить все нужные библиотеки, можете написать в терминале pip install -r requirements.txt. Названия библиотек указаны в requirements.txt.
Также может потребоваться пакет для работы с Jupyter Notebook.

Электромобили (EV) стали важной частью глобального автомобильного рынка, и их продажи продолжают расти. В этом отчете представлены данные о мировых продажах электромобилей с 2010 по 2024 год.
Мировые продажи электромобилей продолжают расти, и прогнозы на будущее остаются оптимистичными. Рост числа моделей, улучшение технологий аккумуляторов и расширение зарядочной инфраструктуры способствуют увеличению продаж. Электромобили становятся все более популярными и доступными, что способствует их дальнейшему распространению на глобальном рынке.
